# Python Task Manager

Aplicação CLI para gerenciamento de tarefas desenvolvida em Python.

## Funcionalidades

- Criação, armazenamento, atualização de tarefas
- Persistência de tarefas em JSON

## Tecnologia utilizada

- Python

## Estrutura do Projeto

- models/
- services/
- main.py

## Como executar

'''bash
python main.py
'''

## Aprendizados

- Criação de Classes
- Separação de Responsabilidade
